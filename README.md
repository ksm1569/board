# 게시판 API
### ✨ 개발 목적 및 개요
대규모 트래픽을 고려한 게시판을 만들어 보며 공부

- 게시판 도메인 CRUD - 카테고리, 게시글, 댓글, 대댓글, 태그, 관리자 공지글 등
- Redis - 캐싱으로 성능 최적화 해보기 + 성능 테스트
- Spring AOP 제대로 사용해보기
- 젠킨스로 배포 자동화 해보기

### 🔨 사용한 프레임웍, 라이브러리 및 툴
- JAVA 17
- Spring Boot
- Mybatis
- Redis
- Jenkins
